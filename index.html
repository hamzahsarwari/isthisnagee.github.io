PROJECTS
  • Epsilon Delta - Github

    This was the first project that I attempted outside of class. I wanted to teach students how to format and attempt epsilon-delta proofs, and to provide them with some resources that will improve their understanding of the definition of limits. This website takes in a simple function and a limit, and returns a correct proof.
    Languages and Frameworks: Python, Flask, HTML5, CSS3


  • Reddit Bots - Github

    A collection of reddit bots that help users by correcting misspellings of Gandhi and informing people of the location, languages spoken, and population of a requested country.
    Languages and Frameworks: Python, Reddit API


  • Math Movie Database - Github

    A port of this database, with a complete overhall in design and functionality. This project has a lot of firsts: the first time I used django, the first time I used a databse, the first time I scraped a website, and the first time I used vim.
